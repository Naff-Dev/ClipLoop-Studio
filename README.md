# NaffDev ClipLoop Studio

**NaffDev ClipLoop Studio** adalah aplikasi desktop berbasis Electron untuk membuat video loop dengan transisi halus dan audio yang sudah digabung otomatis. Aplikasi ini bekerja **offline**, cepat, dan **100% gratis**.

> Jika ada pihak yang memperjualbelikan aplikasi ini, harap lapor ke pengembang.
